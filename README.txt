== Description ==

A lightweight contact us form with a floating icon on all pages at the bottom left or bottom right. Just activate the plugin and set colors of the icon in settings and recipient email address and it is ready to use.

You can either use default form or use your own form by saving shortcode in Form Shortcode field in Contact Us for WP settings.

== Installation ==

1. Install the plugin via the WordPress plugin directory or download the plugin zip file from WP plugins directory and upload `contactuswp.zip` file in WP dashboard -> 'Plugins' page.
2. Activate the plugin through the `Plugins` page in WordPress dashboard.
3. Go to Settings > Contact US for WP > add your email address as recipient email.

== Frequently Asked Questions ==

= Do I need to insert shortcode in my pages? =

No, only activate the plugin and set the settings and it is ready!

= What if I don't recevie emails? =

Consider installing SMTP plugin and set it up.

= What is the format of colors used? =

Hex color codes, i.e #555555
